Images in p5

You can load and display existing images using the following functions:

  • loadImage()
  • image()
  • tint()
  • noTint()
  • imageMode()

You can also read them pixel by pixel. This lets you distort or filter the image.

  • pixels[]
  • blend()
  • copy()
  • filter()
  • get()
  • loadPixels()
  • set()
  • updatePixels()

One of the most important functions in those lists is image(). It can take these five parameters: image(imgVariable,x,y,width,height), but only the first one is required. You can do any of the following:



    image(imgVariable);

    image(imgVariable,100,200); 

    image(imgVariable, 100, 200, imgVariable.width/2, imgVariable.height/2);

Filter

This example shows how to apply filters to images. For more about all the filter options, view the reference.

THRESHOLD
.filter(THRESHOLD, 0.5);
Converts the image to black and white pixels depending if they are above or below the threshold defined by the level parameter. The parameter must be between 0.0 (black) and 1.0 (white). If no level is specified, 0.5 is used.

GRAY
.filter(GRAY);
Converts any colors in the image to grayscale equivalents. No parameter is used.

OPAQUE
.filter(OPAQUE);
Sets the alpha channel to entirely opaque. No parameter is used.

INVERT
.filter(INVERT);
Sets each pixel to its inverse value. No parameter is used.

POSTERIZE
.filter(POSTERIZE,10);
Limits each channel of the image to the number of colors specified as the parameter. The parameter can be set to values between 2 and 255, but results are most noticeable in the lower ranges.

BLUR
.filter(BLUR,1);
Executes a Guassian blur with the level parameter specifying the extent of the blurring. If no parameter is used, the blur is equivalent to Guassian blur of radius 1. Larger values increase the blur.

ERODE
.filter(ERODE);
Reduces the light areas. No parameter is used.

DILATE
.filter(DILATE);
Increases the light areas. No parameter is used.

Blending

This example shows you how to blend two images using the blend() method. This method takes a lot of parameters so be sure to read through the reference page on what each parameter does. The blend modes you can use are:

  • BLEND - linear interpolation of colours. This is the default blending mode.
  • ADD - sum of A and B
  • DARKEST - only the darkest colour succeeds
  • LIGHTEST - only the lightest colour succeeds
  • DIFFERENCE - subtract colors from underlying image.
  • EXCLUSION - similar to DIFFERENCE, but less extreme.
  • MULTIPLY - multiply the colors, result will always be darker.
  • SCREEN - opposite multiply, uses inverse values of the colors.
  • REPLACE - the pixels entirely replace the others and don't utilize alpha (transparency) values.
  • OVERLAY - mix of MULTIPLY and SCREEN . Multiplies dark values, and screens light values.
  • HARD_LIGHT - SCREEN when greater than 50% gray, MULTIPLY when lower.
  • SOFT_LIGHT - mix of DARKEST and LIGHTEST. Works like OVERLAY, but not as harsh.
  • DODGE - lightens light tones and increases contrast, ignores darks.
  • BURN - darker areas are applied, increasing contrast, ignores lights.
